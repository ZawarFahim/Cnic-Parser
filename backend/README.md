# CNIC.io Backend

## Technologies Used

- FastAPI
- MongoDB Atlas
- Google Cloud Vision API
- LangChain
- Gemini 2.5 Flash

## Installation

1. Create virtual environment

python -m venv .venv

2. Activate environment

3. Install dependencies

pip install -r requirements.txt

4. Copy

.env.example

to

.env

5. Add your API keys.

6. Run

uvicorn app.main:app --reload
